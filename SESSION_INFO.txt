====================================

Sessão Conectada: default
Caminho dos Dados: ./waha-sessions/webjs/default

IMPORTANTE:
Esta sessão deve ser preservada para garantir o funcionamento contínuo do bot.
O diretório 'waha-sessions' contém os dados de autenticação do WhatsApp.
Se este diretório for apagado, será necessário escanear o QR Code novamente.

PERSISTÊNCIA:
A persistência é garantida pelo mapeamento de volume no docker-compose.yml:
    volumes:
      - ./waha-sessions:/app/.sessions

Mesmo após rodar 'make build' ou recriar os containers, a sessão será mantida
desde que a pasta './waha-sessions' não seja deletada.
